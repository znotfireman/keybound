# Keybound

My create task for AP Computer Science Principles. Keybound is a keyboard HUD
for gameplay recordings, especially with precision obstacle courses on Roblox.

https://github.com/user-attachments/assets/85e90f8b-5a92-4971-83fc-ec61eb14d4e7

## Usage

```sh
# clone the repository
git clone https://github.com/znotfireman/keybound.git
cd keybound

# (recommended) initialize a virtual environment
python -m venv venv
source venv/bin/activate

# install dependencies
pip install -r requirements.txt

# run the program!
python main.py
```

To use pynput, add Python to your accessibility settings. It's better to be
safe than sorry:
